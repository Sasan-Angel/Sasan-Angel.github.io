Tipsy

SMS Police

This is SMS Police. This app seperates the spam sms messages and put it in different folder on your iOS device official messages app.

By default applications don’t have access to messages that’s why I used “ Message Filter Extension ” that Apple provided to developers to enable them make some rules that decide if a message is spam or not.

I used machine learning in this app so the ml model will get better in deciding which message is spam.

UI/UX Designed by me.
